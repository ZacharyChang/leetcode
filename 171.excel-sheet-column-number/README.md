# Excel Sheet Column Number
-----
## Description
Given a column title as appear in an Excel sheet, return its corresponding column number.

## Example
### Example 1
```
A -> 1
B -> 2
C -> 3
```

### Example 2
```
Z -> 26
AA -> 27
AB -> 28 
```

### Example 3
```
Input: "ZY"
Output: 701
```
