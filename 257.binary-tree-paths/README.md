# Binary Tree Paths
-----
## Description
Given a binary tree, return all ```root-to-leaf``` paths.

## Example
```
Input:

   1
 /   \
2     3
 \
  5

Output: ["1->2->5", "1->3"]

Explanation: All root-to-leaf paths are: 1->2->5, 1->3
```

## Note
A leaf is a node with ```no children```.
