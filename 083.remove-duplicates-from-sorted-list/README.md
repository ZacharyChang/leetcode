# Remove Duplicates from Sorted List
-----
## Description
Given a ```sorted linked list```, ```delete all duplicates``` such that each element appear only once.

## Example
### Example 1
```
Input: 1->1->2
Output: 1->2
```

### Example 2
```
Input: 1->1->2->3->3
Output: 1->2->3
```
